## 이렇게 만든 이유

- **완전 로컬 상태**: 정기구독 백엔드가 없어서 `useState`에 하드코딩 데이터로 동작. 새로고침하면 원래 데이터로 돌아감.
- **토글 스위치**: 활성/일시정지를 직관적으로 전환. 슬라이딩 dot 애니메이션 (Framer Motion `layout` + spring).
- **투명도 변경**: 일시정지된 구독은 `opacity-50`으로 흐리게 표시. 시각적으로 비활성 상태 구분.
- **데모 안내 배너**: 사용자가 실제 기능으로 오해하지 않도록 명확히 안내.

---

## 주의할 점

- **데이터가 하드코딩이에요.** `initialSubscriptions`에 2개의 구독 아이템이 코드에 직접 써 있습니다:
  - 프리미엄 원두 (월 1회, 5박스)
  - 테이크아웃 컵 (월 1회, 10박스)
- 토글을 눌러도 API 호출 없음. `setSubscriptions()`로 로컬 상태만 변경.
- 새로고침하면 토글 상태가 초기화됩니다 (persist 없음).
- "구독 상품 추가하기" 버튼이 `/products`로 이동하지만, 실제로 상품 목록에서 구독을 추가하는 기능은 없어요. 링크만 연결.
- 이 페이지에 사이드바가 없어요. 마이페이지와 달리 단독 레이아웃. 뒤로 가려면 Header나 브라우저 뒤로가기를 써야 합니다.
