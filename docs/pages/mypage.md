# 마이페이지 (`/mypage`)

> 마지막 확인: 2026-03-30 | 파일: `src/app/mypage/page.tsx`

## 한줄 요약
내 정보, 사업자 정보 확인 + 주문내역/정기구독으로 이동하는 허브 페이지.

---

## 스크린 구성

1. **Header**
2. **레이아웃 (사이드바 + 메인)**
   - **왼쪽 사이드바**
     - 아바타 (대표자명 첫 글자 + 그라데이션 원형)
     - 유저 이름 + 회사명
     - 네비게이션 3개:
       - 내 정보 (현재 페이지, 활성)
       - 주문 내역 → `/orders`
       - 정기구독 관리 → `/mypage/subscription`
     - 로그아웃 버튼
   - **오른쪽 메인**
     - **계정 정보 카드** — 이메일 + "사업자 변경" 버튼 (미구현)
     - **사업자 정보 카드** — 상호명, 대표자명, 사업자등록번호, 업종 + "수정" 버튼 (미구현)
     - **회원 등급 카드** — 현재 등급 뱃지 + "누적 구매금액 30만원 이상 시 프리미엄" 안내

## 사용자 흐름

- **어디서 오나**: Header의 유저 아바타 클릭
- **어디로 가나**:
  - 사이드바 "주문 내역" → `/orders`
  - 사이드바 "정기구독 관리" → `/mypage/subscription`
  - 로그아웃 → `/login`

## 사용하는 것들

| 구분 | 이름 | 파일 경로 |
|------|------|----------|
| 컴포넌트 | Header | `src/components/header.tsx` |

| 스토어 | useAuthStore | `src/store/auth-store.ts` |

### API 호출
- `useMe()` → GET `/users/me` — 내 정보 조회
- `useLogout()` → POST `/auth/logout` — 로그아웃

---

## 이렇게 만든 이유

- **사이드바 네비게이션**: 마이페이지 하위에 여러 섹션이 있어서 사이드바로 정리. `layoutId="activeNav"`로 활성 표시가 슬라이드.
- **아바타에 이름 첫 글자**: 프로필 사진 기능이 없어서 대표자명 첫 글자를 그라데이션 원 안에 표시.
- **"사업자 변경"/"수정" 버튼**: UI만 있고 클릭해도 아무 동작 안 함. `useUpdateMe()` 훅은 있지만 연결 안 됨.
- **회원 등급**: `user.grade` 값을 뱃지로 표시. "누적 구매금액 30만원" 기준은 하드코딩된 안내 문구.

---

## 주의할 점

- 로그아웃 흐름: `useLogout()` 뮤테이션 → `useAuthStore.logout()` → `router.push("/login")`. 3단계 순차 처리.
- "수정" 버튼들이 미구현이에요. PATCH `/users/me` 훅은 `queries.ts`에 준비되어 있지만 이 페이지에서 연결하지 않았습니다.
- `useMe()`가 401을 반환하면 Axios 인터셉터가 자동으로 `/login`으로 리다이렉트합니다.
- 사이드바 네비게이션이 `/orders`로 직접 이동 (마이페이지 하위가 아니라 별도 라우트).
- 로딩 중: 스피너, 에러 시: 에러 메시지.
