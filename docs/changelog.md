## [v1.1.0] — 2025-03-29

### 변경
- 로그인 방식: 카카오/네이버 소셜 로그인 → 이메일 + 비밀번호로 변경
- 결제 방식: 토스페이먼츠 연동 → Mock 결제 (즉시 완료 처리)로 변경
- `/onboarding` 페이지 → `/register` 회원가입 페이지로 통합

### 제거

- 네이버 OAuth 연동
- 토스페이먼츠 SDK 연동
